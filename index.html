Self-service password recovery from UIS

You can reset your Raven Password using a token sent via email or text message.
To configure this feature, log into the Raven Password Management app and follow the 'Configure' link under 'Self-service password recovery'.

The Legacy Raven web authentication system is provided and managed by University Information Services.
